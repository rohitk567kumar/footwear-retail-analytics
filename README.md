# Retail Intelligence for a Small Footwear Business Using Data Analytics

## Project Overview

This project applies data analytics techniques to a small footwear retail business using three years of historical sales and inventory data. The objective of the project is to transform manually maintained retail records into meaningful business insights that support better inventory management and demand forecasting.

## Objectives

* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Analyze product and category performance
* Identify fast-moving and slow-moving inventory
* Perform ABC inventory classification
* Develop demand forecasting models using time-series analysis
* Evaluate forecasting accuracy using MAE, RMSE, and MAPE

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Statsmodels
* Google Colab

## Key Features

* Revenue and profit analysis
* Monthly sales trend analysis
* Inventory turnover analysis
* ABC inventory classification
* ARIMA forecasting model
* Forecast performance evaluation

## Forecasting Results

* MAE: 386,979.23
* RMSE: 448,830.89
* MAPE: 11.62%

## Conclusion

The project demonstrates how data analytics can support inventory optimization and demand forecasting in small retail businesses. The findings underscore the practical value of data-driven decision-making in enhancing operational efficiency and informed business planning.
